# VISIÓN DEL PROYECTO

Changocalypse es un videojuego multijugador mobile 3D donde los jugadores controlan changuitos de supermercado equipados con personajes absurdos, compitiendo en partidas rápidas llenas de caos, power-ups y física arcade.

El objetivo es crear una experiencia divertida, accesible, altamente rejugable y optimizada para dispositivos móviles.
